Project Information

Field Value
Name Magnetic Track Parser
Description Magnetic Track Parser is a Java 7 library that can parse magnetic tracks from a bank issued card. All classes are immutable and thread-safe. The standard `toString()` function formats data in a readable form. Validity is enforced by JUnit tests. Maven is needed for a build.
Homepage https://github.com/sualeh/magnetictrackparser
